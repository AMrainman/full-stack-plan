# Week 1 Demo 合集

本目录汇总第 1 周所有关键 demo，每个文件均可独立运行。

## 文件说明

| 文件 | 对应天数 | 运行命令 | 说明 |
|------|----------|----------|------|
| `promise-chain.ts` | day-01 | `npx tsx promise-chain.ts` | Promise 链式调用与错误处理 |
| `event-loop-order.js` | day-02 | `node event-loop-order.js` | Event Loop 执行顺序 |
| `minimal-http-server.ts` | day-04 | `npx tsx minimal-http-server.ts` | 最小 HTTP 服务 |
| `podcast-server.ts` | day-05 | `npx tsx podcast-server.ts` | 带持久化的播客 CRUD |
| `test.sh` | day-06 | `bash test.sh` | 自动测试脚本 |

## 快速开始

```bash
cd daily/week-01/day-06/demo
pnpm install
bash test.sh
```
