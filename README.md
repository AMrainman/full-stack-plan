# full-stack-plan
全栈学习计划

这个仓库的目的是为了学习并进行全栈开发实战，并提供一些参考资料。

## 每日学习计划

使用 `/daily <week> <day> [hours]` 生成每日学习任务。

示例：
- `/daily 1 1` — 生成第 1 周第 1 天（周一）的 2 小时计划
- `/daily 1 6 5` — 生成第 1 周第 6 天（周六）的 5 小时计划

生成的文件位于 `daily/week-XX/day-XX/`，包含 README、任务清单、知识点、demo、学习资源和复盘。

### 快捷命令

- `/today` — 基于最新计划生成下一天（默认 2h）
- `/today 3` — 基于最新计划生成下一天，指定 3h
